# G1T3-frontend

## Installation

cd and npm install:

```bash
cd client2
npm install
npm run dev
```

ESLint & Prettier installation

```bash
Install ESlint Extension on VSCode : vscode:extension/dbaeumer.vscode-eslint
Install Prettier Extension on VSCode : vscode:extension/esbenp.prettier-vscode
cd client2
npm install
```
